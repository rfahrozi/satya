📊 Deskripsi Lengkap Proyek MONEV

🎯 Overview Proyek

MONEV adalah sistem Monitoring dan Evaluasi berbasis web full-stack yang dirancang untuk PT. Kepri (kemungkinan terkait pemerintahan/pemerintahan daerah).

Fitur utama: Multi-tenant (Satker Portal), Dashboard, User Management, Upload laporan, Email notifikasi otomatis, BullMQ job queue, PostgreSQL DB, MinIO object storage, Redis cache/queue.

Tech Stack: Node.js/Express + React/Vite + PostgreSQL + Docker + TailwindCSS + Jest Testing

📋 Penjelasan Hirarki File

Root Level

  • package.json: Backend dependencies (Express, Knex, BullMQ, MinIO, Redis)
  • knexfile.js: Konfigurasi PostgreSQL DB
  • docker-compose.prod.yml: Deployment production

frontend/

  • React SPA dengan Vite, TailwindCSS, React Query
  • Pages: Dashboard, SatkerPortal, Login, UserManagement
  • State management: TanStack Query

src/ (Backend)

  • app.js: Express server dengan API routes + static frontend
  • Multi-tenant middleware
  • Email service + Job queue (BullMQ)

Database

  • PostgreSQL dengan Knex.js migrations
  • Multi-tenant schema
  • Dynamic periods & deadlines

src/app.js (Main Server)

{`require('dotenv').config();
const express = require('express');
const cors = require('cors');
const routes = require('./routes');
const { errorHandler } = require('./middlewares/errorHandler');

const app = express();
app.use(cors());
app.use(express.json());
app.use(express.static('public'));

app.use('/api/v1', routes);

// SPA Fallback untuk React Router
app.get('*', (req, res) => {
    res.sendFile(path.resolve(__dirname, '../public', 'index.html'));
});

app.use(errorHandler);

app.listen(process.env.PORT, () => console.log(\`API on port \${process.env.PORT}\`));`}

npm run migrate:latest

# Tests
npm test
npm run test --prefix frontend
