Roadmap & Todo Pengembangan SATYA

Backend & API (Tahap Selesai)

  • Core API Terstruktur (Express + Knex + Postgres)

    Fondasi utama server, routing, arsitektur Service-Controller, JWT Middleware multi-role (ADMIN, PIMPINAN, SATKER).

    SELESAI

  • Endpoint Laporan (MinIO Object Storage)

    Proses unggah, timpa laporan menggunakan Multer File Upload stream ke server Object Storage MinIO. Download Pre-Signed URL Support.

    SELESAI

  • Fitur Verifikasi & Peringatan Otomatis (BullMQ Worker)

    Integrasi Cron-Job (H-3, H-1, Hari H) untuk scanning Satker yang Belum Upload / Belum Lengkap via Email, dan Job Notifikasi Verifikasi Email jika di-revisi Admin.

    SELESAI

  • Agregasi Dashboard (Sistem Statistik)

    API Agregasi performa SATKER terkait upload, dihitung berdasarkan kalkulator deadline Tanggal 10 tiap bulannya.

    SELESAI

  • Test Coverage (Integration & Unit Tests)

    Uji skenario Test API dari awal end-to-end tercover dengan Jest, terisolasi sempurna terhadap Database Local Migration.

    SELESAI

Frontend (User Interface) - TAHAP SELESAI

  • Setup UI Framework & Styling

    Instalasi dan Konfigurasi UI Framework (Rekomendasi: Vue/Nuxt atau React/Vite/NextJS) dengan Vite/Webpack. Styling modern dengan TailwindCSS atau framework khusus.

    SELESAI

  • Halaman Autentikasi (Login)

    Halaman Form Login UX interaktif, handling request ke `/api/v1/auth/login`, Cookie/Localstorage JWT interceptor Axios.

    SELESAI

  • Dashboard Utama (Role: Pimpinan & Admin PT)

    Menampilkan Tabel/Chart interaktif mengkonsumsi route `dashboard-agregat`. Filter berdasarkan Bulan/Periode Tahun. Menampilkan persentase kepatuhan tiap PN.

    SELESAI

  • Modul Verifikasi Admin (Role: Admin PT)

    Tabel list dokumen terunggah, action download (Presigned URL click), dan modal Verifikasi Form untuk update status Lengkap/Revisi dan input text catatan Admin.

    SELESAI

  • Portal Laporan Satker (Role: Satker PN)

    Halaman utama satker PN yang menampilkan List 28 Laporan Wajib. Label status (Belum Upload, Review, Lengkap). Tombol Browse Dropzone uploader ke route POST /upload.

    SELESAI

  • Manajemen Pengguna (Role: Admin PT)

    Halaman CRUD untuk melihat, mengedit password atau register Satker PN Baru termasuk pengisian Email Resmi masing-masing PN.

    SELESAI
